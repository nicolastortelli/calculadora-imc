# Calculadora de IMC em C

Programa simples que calcula o IMC (Índice de Massa Corporal) e mostra a classificação.

## Funcionalidades
- Entrada de peso e altura
- Cálculo do IMC
- Classificação automática

## Como executar

```bash
gcc imc.c -o imc
./imc
```
